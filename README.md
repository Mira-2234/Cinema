# 🎬 ReelBox – Movie Discovery Platform

ReelBox is a modern full-stack movie discovery platform where users can explore movies, view detailed information, register/login, and manage their own movie collection. The application is built with **Next.js, TypeScript, Express.js, MongoDB, JWT Authentication**, and follows responsive UI/UX best practices.
